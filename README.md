# Strategy Research Archive

一个面向策略研究、线程拆解、聊天截图整理与延展分析的资料库。

这里的文档不是原文搬运，而是按固定规则整理后的中文投研笔记：重点提炼主题、判断方法、应对策略、风险与限制，并补充延展思路。
