## 原文标题

永续合约：小而美的ALPHA之路

## 原文正文整理

文章开头提出：现代统计学和机器学习已经让传统量化交易进入红海，顶级团队追求低衰减、大容量、工程完备的深度策略。小团队如果在大而全的因子挖掘上和大厂硬拼，胜率很低；更现实的方向，是理解加密货币永续合约中的市场结构和机制漏洞。

### 因子失效的底层逻辑

作者认为因子失效不是偶然，而是金融市场作为自适应博弈系统的必然结果。

1. 因子拥挤：从 Alpha 到 Beta 的退化

- 金融市场中的因子本质上是对历史数据中某种盈利模式的拟合。
- 一旦大量资金识别并交易这个模式，交易行为本身会改变市场边界。
- 原本的小市值、高动量等 alpha，会因为参与者过载而退化成 beta，甚至变成拥挤流动性风险。

2. 参数空间相变：从稳定域滑向盲区

- 市场不是稳态系统。
- 低利率、高增长环境下训练出的因子，在加息环境下可能直接失效。
- 均值、方差、协方差等统计特征会漂移，昨天的最优解在新的参数空间里可能不再可行。

3. 分布伪尖锐：过拟合与虚假证据

- 在数万个变量中搜索，总能找到和历史收益高度拟合的因子。
- 看似高预测力的分布，可能只是样本不足、偶然切片或噪声伪装出来的确定性。

作者的结论是：量化因子是在用静态、局部的数学公式捕捉一个动态、自适应、具备社会属性的复杂系统，所以策略会衰减、环境会变、黑天鹅会引入历史数据里从未出现的新维度。

### 为什么永续合约仍然存在结构性 Alpha

作者认为，加密永续合约仍然像一片流动性碎片化的原始丛林，规则复杂、机制异质，为小团队留下了一些未经充分算力污染的微观套利空间。

1. 碎片化市场带来的参数红利

- 加密市场交易所林立，流动性彼此割裂。
- 同一资产在不同盘口之间天然存在定价偏差。
- 各交易所手续费、返佣、深度逻辑、指数规则不同，限制了统一大模型回测的精度，却给小团队做场景级精细化拟合留下缺口。

2. 资金费率中的机制确定性

- 资金费率是永续合约锚定现货指数的核心机制。
- 不同交易所指数成分不同，有的偏向 Binance，有的包含二线平台。
- 当某个平台盘口出现非理性溢价，资金费率计算逻辑会形成强制回归引力。
- 作者认为这种 alpha 更像合约机制驱动，而不是纯历史统计拟合。

3. API 与工程红利

- 一些交易所工程质量弱，API 不完善，甚至不提供 API。
- 小团队可以通过反向推演订单网关、撮合引擎或延迟机制，把平台系统缺陷转化为 alpha。
- 这类机会容量有限，但工程门槛高，半衰期可能更长。

4. Binance 的定价指示效应

- Binance 作为全球流动性中心，其盘口变动常常先于外围市场。
- 小交易所指数更新和盘口反应可能滞后数毫秒甚至数秒。
- 这种 lead-lag 结构为小团队提供信息延迟套利空间。

### 反身性套利与机制博弈

作者认为，小团队的生存之道是钻机制空子，而不是做大容量通用模型。永续合约链条很长且脆弱：

- 项目方发行代币合约可能出错
- 交易所采集指数可能出错
- 做市商摆单可能异常
- 指数、撮合、费率、结算、人工干预之间有大量耦合点

异常事件总能提供套利机会。

### 资金费率与价差的反身性实践

作者举了几个资金费率套利结构：

1. 标准期现资金费率套利

- 买入现货，做空永续，吃正费率。
- 但真实执行时，永续价格可能已经远低于现货，等于先支付一大笔价差成本。

2. 价差优先的快速套利

- 如果足够快，可以在资金费率尚未结算前，利用价差直接出场。
- 价差套利会压低溢价指数，进而影响资金费率本身。
- 这体现了资金费率和价差之间的反身性关系。

3. 单腿二次反身性套利

- 在高正费率市场，提前开空，结算后平仓吃费率。
- 但如果很多人都这么做，结算后“买入平空”的拥挤会把价格打高，费率未必覆盖手续费和滑点。
- 于是可以反过来在结算后立刻开多，吃平空造成的二次反身性上涨。

4. 逃票策略

- 不同交易所资金费率快照扫描时间不同。
- 有些小 CEX 可能存在结算点后短时间内打入订单而不被收取费率的窗口。
- 作者提醒这类机会容量不大，操作不好可能被交易所扣费，且流动性和费率都好的品种不多。

### 跨所资金费率的进阶战场

作者认为，跨所交易是更进阶路径。

- 多市场的近似同品种存在规则差异、费率差异和指数差异。
- 在 CEX 高额资金费率结算时，可以考虑 CEX 开空、DEX 开多。
- 价差与费率互为成本关系，真正机会来自各平台规则不一致和指数差异。

文章最后说，小团队的优势在于不追求大容量，切换成本低，可以在合规灰色地带捕猎。让大机构做模型 alpha，小团队做机制 alpha。

## 评论区与补充

### 1. 预测市场和 CEX 套利也是相邻方向

- `@derrrrrrrq` 提到，预测市场流动性变好后，预测市场和 CEX 之间的套利也可能值得挖掘。
- 作者回复：这块学问很大，自己也还在研究。

### 2. 交易所资金费计算公式差异本身就是 alpha 来源

- 有评论补充：交易所太多，各所资金费计算公式不同，会天然导致一些 alpha 存在。
- 但资金分散和跨所风险会限制收益率。

### 3. 实盘基础设施是持续门槛

- 有评论提到跨所资金费套利的 Rust 实盘 infra，并指出交易所 API 经常更新。
- 这强化了原文的工程门槛：这类 alpha 不只是发现逻辑，还要长期维护执行系统。
