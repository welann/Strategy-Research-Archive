## 主题

这篇内容真正讲的是：**分散投资不是买很多 ticker，而是让组合拥有真正低相关的收益来源，从而降低组合波动、减少 volatility drag，并释放风险预算。**

作者用半导体集体下跌但自己没亏钱的例子，说明：

- 全仓半导体不是分散，而是做多同一个 factor
- 组合保护来自 correlation matrix 上的分散
- 少赚不亏钱不是怂，而是复合收益数学

## 核心机制

### 1. 买很多只同一板块股票，不等于分散

作者直接点出：

- `NVDA / AVGO / AMD / MU / MRVL / INTC`

看起来是六只票，实际上是一个 trade：

- 做多 `AI capex cycle`
- 做多 semiconductor beta

当 risk-off 来的时候，这些资产相关性可能直接跳到 `0.9`。  
涨的时候你以为自己分散，跌的时候才发现所有仓位都在同一个门口排队出逃。

### 2. 真正的分散来自相关性矩阵

作者用组合方差公式解释：

```text
σ_p² = w₁²σ₁² + w₂²σ₂² + 2w₁w₂ρσ₁σ₂
```

关键变量是 `ρ`：

- `ρ = 1`：没有分散收益
- `ρ = 0`：组合波动显著下降
- `ρ < 0`：组合波动会因为对冲项变负而进一步下降

所以分散的核心不是资产数量，而是收益来源之间的相关性。

### 3. 降低波动本身就能提高长期复合收益

作者引用：

```text
g ≈ μ - σ²/2
```

这说明同样的 expected return，如果波动更低，长期几何收益更高。

这就是 volatility drag：

- 波动越大，复合收益被拖得越狠
- 控制下行，对长期 CAGR 的贡献可能比追求单年高收益更大

### 4. 再平衡可以从横截面波动中挤收益

作者提到 `Claude Shannon` 的思想实验：

- 一个 expected return 为零但波动大的资产
- 搭配现金机械再平衡
- 长期可能产生正收益

机制是：

- 涨了减
- 跌了加

减半导体、加其他低相关 sleeve，不只是择时，也是纪律性再平衡。  
它可以成为独立于方向判断的 alpha source。

### 5. 低波动组合释放风险预算和 optionality

如果组合波动从 `30%` 降到 `18%`，就有更多风险预算做其他事情：

- 加低相关 sleeve
- 买期权做 convexity
- 留现金等真正 fat pitch

全仓 momentum 的人没有这个 optionality，因为风险预算每天都被同一 narrative 占满。

## 作者自己的仓位结构

作者没有完全离开半导体，而是降低半导体权重，并加入：

- `staples`
- `utilities`
- 能源
- long-duration treasury

这些 sleeve 和半导体的驱动不同：

- 防御板块
- 公用事业
- 地缘与油价 catalyst
- risk-off convexity

组合目标不是在牛市里跑赢最极端动量仓，而是在拥挤 trade unwind 时还能留下足够多本金和风险预算。

## 风险与限制

### 1. 这套方法会在牛市里跑输纯动量

作者自己也承认，这是结构性 trade-off。

如果半导体一直单边涨，全仓半导体会赢。  
但问题是长期账户不是看某一刻峰值，而是看路径和回撤之后剩下多少。

### 2. 相关性不是静态的

风险状态下，相关性会跳变。  
平时低相关的资产，在极端流动性冲击中也可能一起跌。

所以 correlation matrix 不是永久答案，只是比 ticker 数量更接近真实风险。

### 3. 半导体内部不能完全粗暴合并

评论区补充得很重要：

- `HBM` 和消费级存储不是同一逻辑
- 供给结构、定价权、产能约束不同

所以不能把半导体全部视为一个完全同质 factor。  
正确做法是先按驱动因子分层，再看组合相关性。

### 4. 再平衡规则需要事先定义

主帖强调再平衡重要，但没有给出具体阈值。

真正落地时需要提前规定：

- 多大偏离触发再平衡
- 按时间再平衡还是按权重再平衡
- 是否考虑税费、滑点和流动性

## 扩散分析 / 延展思路

### 1. 可以把每个仓位先归因到 factor，而不是 ticker

例如：

- AI capex
- semiconductor beta
- rate sensitivity
- energy geopolitics
- defensive consumption

然后再看自己到底有几个真正独立的收益来源。

### 2. 组合管理要同时看收益和退出门

拥挤动量 trade 的问题不是涨不上去，而是所有人退出同一扇门。

所以风险管理要问：

- 谁和我拿着同一笔交易
- 他们会在什么情况下同时卖
- 盘口能不能接住

### 3. “少赚不亏钱”可以翻译成更精确的目标

不是保守，而是追求：

- 更低波动
- 更小回撤
- 更高几何收益
- 更长生存时间

这比“我不想亏钱”更接近可执行的组合目标。

## 一句话结论

**这篇内容的核心是：真正的分散不是买很多半导体 ticker，而是用低相关收益来源、纪律性再平衡和风险预算管理，把一个 crowded momentum trade 改造成能长期复合的组合。**
